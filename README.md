# VitalVoice - AI Clinical Assistant 🌌🩺👤

VitalVoice is a high-fidelity AI medical coaching agent designed for the **Murf AI Voice Hackathon**. It features a cinematic dashboard, real-time multilingual sync, and deep medical intelligence powered by Mistral AI.

## 🚀 Quick Start

### 1. Clone & Install
```bash
git clone https://github.com/prashantsingh85699-gif/MYAIVOICEAGENT.git
cd MYAIVOICEAGENT
```

### 2. Configure Keys
Create a `.env` file in both `/backend` and `/frontend` with your respective API keys.

### 3. Launch Backend
```bash
cd backend
npm install
npm run dev # Starts on http://127.0.0.1:9091
```

### 4. Launch Frontend
```bash
cd frontend
npm install
npm run dev # Starts on http://127.0.0.1:3333
```

## ✨ Features
- **Misty Persona**: A warm and empathetic AI medical friend.
- **Multilingual Sync**: Seamless English/Hindi voice and text (Strict Devanagari script for Hindi).
- **Medical Dashboard**: Responsive HUD with chat auto-scroll, brain status, and clinical insights.
- **Interactive UI**: Talking avatar with bar-based lip-sync and glowing 'VitalCore' status orb.
